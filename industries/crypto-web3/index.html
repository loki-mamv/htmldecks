Frequently Asked Questions

Can I host HTML Decks presentations on IPFS?

Yes. HTML files can be hosted on IPFS, Arweave, or any decentralized storage. A single HTML file with embedded assets is perfect for permanent, censorship-resistant hosting.

Can I include live blockchain data in presentations?

Since HTML supports JavaScript, you can embed live data from APIs — token prices, TVL, on-chain metrics. For reliability during presentations, consider static snapshots with timestamps.

How do crypto projects present tokenomics?

Use our templates for pie charts showing token allocation, vesting schedules, and supply mechanics. The comparison layouts work great for showing different token utility scenarios.

Do HTML Decks work for DAO governance proposals?

Yes. Create structured proposals with budget breakdowns, timeline, success metrics, and team qualifications. Share as a link that any DAO member can review on any device.
