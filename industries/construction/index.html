Frequently Asked Questions

Can I use HTML Decks for safety training on job sites?

Yes. HTML files work on any device — tablets, laptops, even smart TVs. No internet needed after loading. Perfect for toolbox talks, safety orientations, and OSHA compliance training.

How do GCs use HTML Decks for project proposals?

General contractors create proposal decks with project approach, timeline, budget overview, team qualifications, and relevant past project photos. The professional format helps differentiate in competitive bids.

Can I include project photos and progress documentation?

Yes. HTML supports high-resolution images. Include drone photos, progress shots, before/after comparisons, and project completion documentation.

Do HTML Decks work on job site tablets?

Yes. Any tablet with a browser can display HTML files. They work offline once loaded and look great on any screen size. No special apps or software required.
