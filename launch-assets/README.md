## Asset Specifications

### Screenshots
- Format: PNG
- Max file size: 5MB each
- Recommended: 2560x1600px (retina)
- Show actual product in use
- Clean, professional framing

### GIF
- Format: GIF
- Max file size: 15MB
- Duration: 15-20 seconds
- FPS: 20-30
- Optimized for web

### Video
- Format: MP4 (H.264)
- Max file size: 100MB
- Duration: 30-60 seconds
- Resolution: 1920x1080 or higher
- Include captions

### OG Image
- Format: PNG or JPG
- Size: 1200x630px exactly
- File size: <1MB
- Includes: Logo, tagline, key benefit
